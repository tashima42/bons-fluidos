# bons-fluidos
API Docs: https://documenter.getpostman.com/view/13233153/2sA3rwMuDD

Backend para o aplicativo Bons Fluidos

## Desenvolvimento

Database: SQLite

```
export DATABASE_PATH=./bons_fluidos.db
export JWT_SECRET=secret
export PORT=3000
go run .
```

Frontend
```
cd front-web
npm rum dev
```
